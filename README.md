# React Profile Page App

A simple React profile dashboard built with **Vite**.  
This project displays a user profile including stats, personal information, and account details using reusable React components.

---

## Features

- Profile header with avatar, name, and username
- Statistics cards (XP, Streak, Badges)
- Personal information section
- Account details section
- Component-based React structure
- Fast development with Vite

---

## Technologies Used

- React
- Vite
- JavaScript (JSX)
- CSS

# Getting Started

Follow these steps to run the project locally.

Install dependencies

- npm install

# Run the development server

- npm run dev

# Open your browser

Visit the following URL to view the app:

- http://localhost:5173
